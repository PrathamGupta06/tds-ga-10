# TDS RevealJS Presentation

This repository contains a RevealJS presentation showcasing data science and financial mathematics concepts.

## Features

- **Email Contact**: 23f3001608@ds.study.iitm.ac.in
- **Markdown Slides**: Multiple slides written in Markdown format
- **Animated Elements**: Fragment animations for progressive disclosure
- **Code Samples**: Python and JavaScript examples with syntax highlighting
- **Mathematical Equations**: Financial formulas (PV, FV, CAGR, Sharpe Ratio)
- **Speaker Notes**: Comprehensive notes for each slide

## Viewing the Presentation

The presentation is deployed on GitHub Pages and can be accessed at:
[View Presentation](https://prathamgupta06.github.io/tds-ga-10/)

## Local Development

To view the presentation locally, simply open `index.html` in a web browser.

## Navigation

- Use arrow keys or swipe to navigate between slides
- Press `S` to open speaker notes view
- Press `F` for fullscreen
- Press `ESC` for slide overview
