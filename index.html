Financial Mathematics

Present Value (PV):
\[ PV = \frac{FV}{(1 + r)^n} \]

Future Value (FV):
\[ FV = PV \times (1 + r)^n \]

Compound Annual Growth Rate (CAGR):
\[ CAGR = \left(\frac{FV}{PV}\right)^{\frac{1}{n}} - 1 \]

These are fundamental financial formulas used in investment analysis. PV = Present Value, FV = Future Value, r = rate of return, n = number of periods. Explain each formula with real-world examples.

JavaScript Example


// Calculate financial metrics
function calculateCAGR(initialValue, finalValue, years) {
    return Math.pow(finalValue / initialValue, 1 / years) - 1;
}

// Example usage
const cagr = calculateCAGR(1000, 1500, 3);
console.log(`CAGR: ${(cagr * 100).toFixed(2)}%`);

Statistical Formulas

Mean (Average):

\[ \mu = \frac{1}{n}\sum_{i=1}^{n} x_i \]

Standard Deviation:

\[ \sigma = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i - \mu)^2} \]

Sharpe Ratio (Risk-Adjusted Return):

\[ S = \frac{R_p - R_f}{\sigma_p} \]
These statistical formulas are crucial for financial analysis. The Sharpe Ratio helps evaluate risk-adjusted returns of investments. Higher Sharpe Ratio indicates better risk-adjusted performance.
